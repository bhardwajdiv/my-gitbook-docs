# Connecting GitHub to GitBook

The foundational power of GitBook relies on linking your cloud repository directly to the hosting rendering layer.

## Authentication Framework Mechanics
To safely open up a sync line between your systems, you must configure a secure authorization pathway.

### 1. Integration Authorization
Open the specific space settings dashboard, select the integration tab options, and initialize the **GitHub Sync** extension module.

### 2. OAuth Handshake Setup
Authorize the official GitBook OAuth engine token request to grant explicit security permissions over your target repository scope (`bhardwajdiv/my-gitbook-docs`).
