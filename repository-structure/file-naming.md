# File Naming Conventions

Consistent nomenclature prevents broken hyper-links and ensures long-term framework maintainability across multiple tracking teams.

## Production Filename Standards
Adhere strictly to these core file-naming rules across all document nodes:

### Lowercase Character Strings
Always enforce standard lowercase character values across all names (e.g., use `file-naming.md` instead of `FileNaming.MD`).

### Kebab-Case Spacing Strings
Never inject raw empty spacing tokens into file nomenclature. Replace empty word gaps cleanly with standard dash (`-`) characters to prevent browser URL encoding corruption (`%20`).
