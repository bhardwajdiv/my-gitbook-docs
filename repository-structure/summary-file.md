# The Role of SUMMARY.md

The `SUMMARY.md` file is the master configuration file for GitBook integration. It defines how text links convert into user-friendly sidebar items.

## Creating Nested Navigation
To avoid a flat, unorganized list of links (which looks unprofessional), use indentation. 
* Indent child items by using 2 or 4 spaces underneath a parent bullet point.
* Use Markdown headings (`##`) to create bold, non-clickable category dividers.
