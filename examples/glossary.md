# Documentation Glossary

Learn common DevOps and technical writing terms used throughout the GitBook and GitHub synchronization architecture.

## Automated Webhook
An automated webhook is a server-to-server API push notification triggered by specific event listeners. In this ecosystem, pushing code to GitHub causes a webhook to send a payload securely to GitBook, initiating a live compilation pass instantly.

## Hierarchical Tree Mapping
Hierarchical tree mapping is the structural process of sorting individual document nodes into precise parent-child relationships. This prevents flat navigational structures and creates a fluid user experience using logical category divisions.

## Markdown Token Architecture
Markdown tokens are plain-text characters used to dictate front-end layout rendering metrics without requiring raw HTML elements. This includes using asterisks for emphasis weights or hashes for section indexing rules.
