# SEO & Accessibility

Documentation must remain inclusive and easily searchable across public search engine indexing grids.

## Universal Design Requirements
Ensure all digital written assets comply with these two entry accessibility rules:

### Alternative Media Descriptive Strings
Every visual image element must include an explicit alternate text property value description inside the markdown brackets: `![Descriptive Alternative String Text](image.png)`.

### Header Chronology Order
Never skip header sizes out of sequence (e.g., do not place an `### H3` block directly under an `# H1` heading without an intermediary `## H2` block). This ensures screen readers can parse page structures sequentially.
